# Internship LMS Backend

A backend REST API for an Internship Learning Management System.

## Features
- User Authentication (JWT)
- Role-based access (Admin & User)
- Course creation
- Course enrollment
- Admin dashboard APIs

## Tech Stack
- Node.js
- Express.js
- MongoDB
- Mongoose
- JWT
- bcrypt

## API Modules
- Auth (Register / Login / Profile)
- Admin (Courses, Enrollments)
- User (Enrollments, Courses)

## How to Run
1. npm install
2. create .env file
3. node index.js
